# koishi-plugin-chatgpt

## 使用教程

1. 通过 <https://chat.openai.com/chat> 注册并登录。
2. 打开浏览器开发者工具，切换到 Application 标签页。
3. 在左侧的 Storage - Cookies 中找到 `__Secure-next-auth.session-token` 一行并复制其值。
4. 将值填写到 `token` 字段并启用插件。
